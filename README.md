# Rock-paper-scissors
An easy rock paper scissors game in python 

A simple Python command-line game where the player competes against the computer in Rock-Paper-Scissors.

## Features
- User vs. Computer gameplay
- Randomized computer choices
- Input validation (ensures user enters "rock", "paper", or "scissors")
- Tracks the winner of each round
- Replay option to play multiple times

## How to Play
1. Clone or download the repository.
2. Open the Python file in your IDE or terminal:
```bash
python rock_paper_scissors.py
